# Personal Portfolio Website

A clean, multi-page personal portfolio website built entirely with **HTML** where i'm showcasing my education, experience, projects, hobbies, strengths, and areas of growth as an aspiring AI/ML Engineer.

👩‍💻 **Author:** [Layba Akram](https://github.com/LaybaAkram)

---

## About This Project

This portfolio is a static, multi-page website designed to introduce me as a developer and student, currently pursuing a Bachelor's in AI. It's built as a pure HTML project, with lightweight inline styling and no external frameworks, stylesheets, or JavaScript.

The goal of this project was to practice structuring a real multi-page site with consistent navigation and clean, semantic markup, a solid HTML foundation before layering in CSS and JavaScript later.

---

## Pages

| Page | File | Description |
|---|---|---|
| Home | `index.html` | Landing page with a short introduction |
| About | `about.html` | Personal details, background, and future goals |
| Education | `education.html` | Academic history in table format |
| Experience | `experience.html` | Work experience and featured projects |
| Hobbies | `hobbies.html` | Personal interests outside of academics |
| Strengths | `strengths.html` | Key personal and professional strengths |
| Weaknesses | `weakness.html` | Self-aware growth areas and how I'm improving them |

---

## Tech Stack

- **HTML5 only** : semantic structure and inline styling, no external CSS or JS files
- **No frameworks, no build tools** : a deliberately lightweight, dependency-free static site
